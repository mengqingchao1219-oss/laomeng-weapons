## 🚀 使用示例

### 场景1：写口播文案
```
用户："我在郑州开胡辣汤店6年了，老顾客认但年轻人不来，想拍短视频"

编导之刃输出：
- 主判断：一碗好胡辣汤和一碗糊弄的之间，差的不是配方，是6年没省过一天料
- 完整口播：五感写法全开，从牛骨汤底辛香写到额头微微冒汗
- 拍摄建议：开头给汤特写→中段切后厨空镜→结尾往前凑半步
- 评论引导："你喝胡辣汤最看重什么？汤底、辣度、还是牛肉量？"
```

### 场景2：选题诊断
```
用户："我做实体店的，不知道拍什么好"

选题之秤输出：
- 四象限各5题，共20个选题方向
- 每题带人群画像+切入点+传播潜力评估
- 优先做：XX题（原因：破圈+信任双收割）
```

### 场景3：一句话金句
```
用户："我想说清楚'老板该不该出镜'这件事"

金枪之锤输出：
【核心矛盾】明明老板出镜最能建立信任，却最不敢出镜
【一句话判断】你的脸不是缺点，是信任入口
【传播钉】你没讲明白之前，别人不会有耐心慢慢了解你
```

## 📐 设计哲学

- **不压缩**：原文所有智慧、示例、思维链全部保留
- **纯文本**：Markdown格式，AI直接可读，不用Lisp/JSON
