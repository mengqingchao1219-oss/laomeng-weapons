## 👤 谁在用

- **实体商家老板**（餐饮/美业/教培/零售）— 建档之翼梳理定位 → 编导之刃出成片 → 配音之翼出音频
- **内容创作者** — 选题枯竭？编导之刃十步工作流跑一遍
- **短视频操盘手/代运营** — 双账号策划案+定制之炉给客户出完整方案
- **AI提示词爱好者** — 近万行纯文本skill，研究专业级系统提示词的最佳样本

## 🚀 使用示例

```
用户："我开了个店，想做短视频但不知道拍什么"

建档之翼：6模块访谈 → 输出客户档案

配音之翼：成片文案 → TTS脚本 → MP3
```

### 场景2：一句话出片
```
用户："我在郑州开胡辣汤店6年，老顾客认但年轻人不来"
编导之刃：一刀从判断切到成片——主判断+口播+拍摄建议+评论引导
```

## 📐 设计哲学

- **不压缩**：原文所有智慧、示例、思维链全部保留
- **纯文本**：Markdown格式，AI直接可读，零二进制依赖
- **自包含**：每个Skill独立可运行，拷到哪用到哪
- **有联邦**：编导之刃规则14包含完整的13把武器调度表
- **有基因**：已注入Humanizer-zh去AI痕迹+旧版口语基因+10个博主标杆基因卡+5条自动思考外挂
